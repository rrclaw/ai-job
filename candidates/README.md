# candidates/

每个候选人一个目录。**除 `example/` 外全部被 .gitignore 排除，个人数据永不进入公开仓库。**

目录内标准产物（对应 SKILL.md 七阶段）：

```
<name>/
  jd/                 # ① 目标岗位 JD 原文与解析
  evidence.md         # ② 素材库：全部可用素材，每条带来源和口径
  positioning.md      # ③ 人岗匹配矩阵 + 一条主线叙事
  output/             # ④ 简历 md/pdf/docx + ⑥ 面试方案
  tracker.md          # ⑦ 投递跟踪与复盘
```

`example/` 是虚构人物「林深」的脱敏演示，所有数据均为示例。
